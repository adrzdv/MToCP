# MToCP (Мобильный терминал сбора контролируемых параметров)
#### Описание
Мобильный терминал сбора контроллируемых параметров.
Проект мобильного приложения для сбора информации по итогам проверок пассажирских поездов.
Используемый стек:
Java/Kotlin, MVVM, SQLite, Room, Retrofit, Jetpack Compose

_NOTE:_ Приложение не хранит персональные данные.

#### Минимальные системные требования:
API 26 (Android 8.0+)
NFC module
*********

##### Реализовано:
* Авторизация пользователей;
* Полный процесс ревизии пассажирского поезда;
* Добавлены основные справочники, необзодиые для работы;
* Реализована возможность фотофиксации нарушения с последующим наложением временной метки;
* Формирование zip-архива с фотоматериалом;
* Добавлена возможность передачи данных между устройствами в условиях отсутствия покрытия мобильной сети;
##### В стадии разработки:
* Поцесс проверки багажных и почтовых вагонов;
* Логика и процесс проверки пунктов продаж
* Видеофиксация нарушения с наложением временной метки на видеоряд;
