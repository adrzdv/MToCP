# MTOCP — Мобильный терминал сбора контролируемых параметров

MTOCP — это Android-приложение для мобильной фиксации результатов технических проверок железнодорожных составов, формирования актов и последующей передачи данных в backend-систему.

Приложение работает в условиях ограниченного или нестабильного подключения и поддерживает оффлайн-сценарии сбора данных.

---
## Основной функционал
- Сбор данных по проверкам
- Работа с мультимедиа
- NFC передача данных
- Формирования отчетности

### Системные требования
- Android 10 +
- NFC required

### UI / UX
- Полностью UI на Jetpack Compose (проведена миграция с XML View)
- Реактивное состояние через State / MutableState
- Разделение UI и бизнес-логики (MVVM)
- Диалоговые окна (ввод данных, подтверждения, результаты)
- Динамическое обновление экранов
### Бизнес-логика
- Работа с domain-моделями (PassengerCar, Violation, Worker и др.)
- UseCase слой для бизнес-операций
- Поддержка сложных сценариев состояния (State holders)
### Сетевое взаимодействие
- Retrofit API layer
- Работа с backend сервисами
- Обработка ошибок запросов
- Асинхронные операции (coroutines)
### Архитектура
- MVVM (ViewModel + State)
- Clean Architecture (частично/эволюционно)
- Разделение на слои:
    - UI (Compose)
    - Presentation (ViewModel)
    - Domain (UseCases)
    - Data (Repository implementation)
